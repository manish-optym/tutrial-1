# Setup Instructions for Anniversary Website

## Required Assets to Download

### 1. Audio File (Om Shanti Om Theme)
- **File Name**: `om-shanti-om-theme.mp3`
- **Location**: Place in `public/audio/` folder
- **Instructions**: 
  - Download the romantic Hindi theme song "Om Shanti Om" from a legal source
  - Rename it to `om-shanti-om-theme.mp3`
  - Place it in the `public/audio/` directory
  - The music player will automatically use this file

### 2. Doodle Couple Images
- **Location**: Place in `public/images/` folder
- **Instructions**:
  - Download 12+ romantic doodle couple images (PNG or JPG format)
  - Name them as: `Gallery-img-1.jpg`, `Gallery-img-2.jpg`, etc.
  - Place them in the `public/images/` directory
  - The gallery will automatically display these images

### 3. GIF Assets (Already Copied)
- The GIF files from loveme-app have been copied to `public/assets/`
- Files include: `think.gif`, `sadface.gif`, `plz.gif`, `attitude.gif`, `cry.gif`, `cute.gif`, `loveme.gif`, `thanks.gif`

## Notes
- The website will work without these files, but the music and gallery images will be missing
- You can use any romantic Hindi audio file and rename it to `om-shanti-om-theme.mp3`
- For doodle images, search for "romantic couple doodle" or "anniversary couple illustration" on free image sites
